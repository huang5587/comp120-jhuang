1. Introduction

https://jordan-marsh.herokuapp.com is a web app that sends and recieves data related to a ride-sharing app not disimilar to uber. If given a vehicle or passenger username, the website will return its corresponding entry from its respective database. If the website is sent data of a specific vehicle or passenger, it will add that entry to its respective database .


2. Methodology

In order to pen-test the website I primarily used curl to send payloads into the website.

3. Abstract of Findings

In my research I have found that malicious agents can exploit the website to 1. Access and manipulate the internal database and 2. Store code in the website to re-direct users to another URL. Manipulation of the database presents a concerning privacy issue as user records can be freely accessed. Redirecting off the web-site can be a security issue as users can be redirected to insecure and malicious websites that can further exploit the user.
